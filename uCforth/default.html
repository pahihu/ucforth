uCforth

Coldfire Quickstart

Download the binary file ucf and the script file forth. Make them both executable on your coldfire uClinux system, type in forth and your away. If you only feel relaxed if forth comes back with ok type ok when the prompt appears.

The binary has been tested on a 5272 and a 5407 system. It should work on all coldfire uClinux systems. Feedback would be welcome.

On line code browse

uCforth uses HTML files to generate uCforth. Use the following links to look at the source at your leasure. The source is in ANSI forth so it is a good place to go if you want to see what forth code looks like.

Mirror

Download the zip file and unzip to the desired location. The default action is to unzip a directory uCforth. If mirroring copy the zip file into the directory uCforth so others can down load the zip file from the mirror. The entry page to link to is default.html.
